SkillTrack — это веб-платформа, созданная с использованием библиотеки Flask, предназначенная для взаимодействия между учениками и учителями.
Система позволяет:
 • Учителям — создавать и выдавать задания, отслеживать прогресс учеников и выставлять оценки.
 • Ученикам — получать задания, выполнять их и отправлять результаты на проверку.

Платформа упрощает учебный процесс, делает коммуникацию между учителем и учеником более удобной и структурированной, а также помогает систематизировать процесс оценки знаний.
